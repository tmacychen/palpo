# Palpo Admin UI 计划文档

## 产品概述

Palpo Admin UI 是一个基于 Leptos WASM 框架的嵌入式管理界面，为 Palpo Matrix Homeserver 提供可视化的管理功能。

## 技术选型

- **前端框架**: Leptos 0.7+ (Rust WASM)
- **UI 风格**: Ant Design + 深色主题 + 玻璃态
- **部署方式**: 嵌入式 WASM
- **样式**: Tailwind CSS

## 功能模块

### 已实现 (API 可用)

| 模块 | 功能 |
|------|------|
| 用户管理 | 列表/搜索/编辑/停用/密码重置/设管理员/影子封禁/whois |
| 设备管理 | 设备列表/删除/批量删除 |
| 房间管理 | 列表/搜索/详情/成员/封禁/删除 |
| 联邦管理 | 服务器列表/详情/重置连接 |
| 服务器信息 | 版本查询 |
